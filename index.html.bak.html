Front-end Code

It’s paramount that the code running a site is light-weight, elegantly written, and accessible to all users and platforms. My passion resides in standards-based design, HTML, CSS (by way of Sass), DRY coding principles, and the utilization of pattern libraries.

Web Design

Great web design means understanding the user's needs and coupling that with a beautiful, useable interface. I believe beauty is nothing without clear content, which is why I design interfaces that speak not only to beauty, but also interfaces that tell a story.

User Experience

Navigating a web site should not feel like a scavenger hunt. I determine the user’s needs through user testing and research. I then use that information to provide the best, most informative, and painless experience on the website.
